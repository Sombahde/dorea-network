# WHAT IS DOREA

Dorea Network is a governed digital grace network that coordinates:
- prayer
- encouragement
- benevolence
- sponsorship
- support
- follow-up care
- acts of mercy

through:
- invitation-based participation
- contract-driven orchestration
- governed runtime execution
- trusted community interaction

Dorea is:
- the agent of mercy
- the digital continuity layer of the community
- always available

The runtime coordinates acts of grace.

The community gives them meaning.

---

# THE SIMPLE INTERACTION MODEL

The entire system is built around a simple interaction pattern:

```plaintext
I need grace
        ↓
The network responds
```

Everything else exists to support:
- continuity
- coordination
- observability
- trust
- participation
- governance

# THE CORE EXECUTION MODEL

Every capability in Dorea executes through the same runtime lifecycle:

```plaintext
Moment
  ↓
Observe
  ↓
Orient
  ↓
Decide
  ↓
Act
  ↓
Telemetry
```

This execution pattern is universal.

There are no alternate execution paths.

Every interaction is treated as:
- a bounded moment of grace
