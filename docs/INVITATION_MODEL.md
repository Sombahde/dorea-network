# DOREA NETWORK — INVITATION_MODEL.md

## Purpose

This document defines the invitation doctrine and trust propagation architecture for Dorea Network.

The Dorea Network grows through:
- relational trust propagation

NOT:
- advertising
- viral mechanics
- public acquisition funnels
- algorithmic engagement optimization
- mass onboarding campaigns

Dorea exists around a core human reality:

> People often struggle in private moments of isolation before those struggles become publicly visible.

Because of this:
- invitation must feel safe
- onboarding must feel relational
- participation must remain bounded
- trust expansion must remain intentional

The invitation model exists to preserve:
- quiet continuity of grace
- relational trust boundaries
- dignity during vulnerable moments
- safe entry into community support

If implementation conflicts with this document:
- THIS DOCUMENT IS CORRECT
